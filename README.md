## Next.js + TypeScript App

### 프로젝트 소개
- Next.js를 이용한 SSG 기반 프로젝트로 각종 영화 정보를 제공합니다.
- TMDB API를 사용하여 영화 정보 데이터를 가져왔습니다.
- [Next Movies 바로가기](https://next-movies-kwakhyun.vercel.app/)

### 주요 기능
- 현재 상영 영화 조회
- 개봉 예정 영화 조회
- 최고 평점 영화 조회

### 사용 스택
- TypeScript
- React
- Next.js

### 프로젝트 생성
```bash
npx create-next-app --typescript
```
